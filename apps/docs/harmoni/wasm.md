# harmoni-wasm (JS/TS)

`harmoni-wasm` is the WebAssembly adapter for the engine. It's the only crate
that uses `wasm-bindgen` and `tsify`: mirror types shadow the `harmoni-core`
structs, derive the TypeScript/ABI machinery, and convert at the boundary. The
result is a typed module the browser can call directly.

::: tip Building the package
The package is generated from Rust and is not tracked in git. Rebuild it with
`pnpm run build:wasm`, which writes `crates/harmoni-wasm/pkg/`. Apps consume it
via a pnpm workspace link (`"harmoni-wasm": "workspace:*"`).
:::

## Initialising

The module must be initialised once before any call:

```ts
import init, { generate_palette } from "harmoni-wasm";

await init();
const palette = generate_palette("#3b82f6", 0, 0);
```

## Exported functions

```ts
generate_palette(hex: string, light_padding: number, dark_padding: number): Palette

generate_palette_with_lightness(
  hex: string, lightness: number[],
  light_padding: number, dark_padding: number,
): Palette

generate_palette_pair(
  hex: string, light_lightness: number[], dark_lightness: number[],
  light_padding: number, dark_padding: number,
): PaletteSet

generate_palette_with_light_padding(hex: string, light_padding: number): Palette

generate_neutral_ramp(white: string, black: string, tint: TintMode): Palette

derive_soft_neutrals(brand: string, softness: number): SoftNeutrals

tint_neutrals(white: string, black: string, source: string, strength: number): SoftNeutrals

get_contrast_rating(bg: string, fg: string): ContrastResult
```

Each maps onto a `harmoni_core::api` function (see
[harmoni-core](/harmoni/core)). Errors are thrown as `JsError`.

## Types

All types are generated by `tsify`, so the package ships full TypeScript
definitions. The main shapes:

```ts
type Palette = {
  swatches: Swatch[];
  lightness_curve: number[];          // length 10
  max_recommended_light_padding: number;
  max_recommended_dark_padding: number;
  note: string;
};

type Swatch = {
  l: number; c: number; h: number;
  label: SwatchLabel;
  hex: string;
  rgb: Rgb;
  oklch: string;
  best_foreground: string;
  foreground_source: ForegroundSource;
  contrast_result: ContrastResult;
};

type ContrastResult = { ratio: number; display_ratio: string; rating: string };
type PaletteSet = { light: Palette; dark: Palette };
type SoftNeutrals = { white: OklchTriple; black: OklchTriple };

type TintMode = "Inherit" | "Achromatic";
type ForegroundSource =
  | "Step900" | "Step50" | "SoftWhite" | "SoftBlack" | "PureWhite" | "PureBlack";
type SwatchLabel = { Number: number } | { Name: string };
type Rgb = { r: number; g: number; b: number };
```
